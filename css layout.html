Single-Column

See the Pen Untitled by student147806 (@student147806) on CodePen.

Here we used CSS Grid to layout a single column, three-rowed Home page. We also used a details tag to condense the two paragraph article into a dropdown menu. By setting the site's display to grid, we can set the size for each "row" or section we used. I set the header to 48px, and the footer to 75px. By setting the article to auto, it forces the footer to the bottom of the page.

Split Screen

See the Pen Split Screen by student147806 (@student147806) on CodePen.

Here, we created a split screen "Get Started" page very similar to the dropbox login. We used CSS Grid to align the columns, and flex to align content within each box. We used modifiers to distinguish the 2 boxes, and added transitions for both the heart and "Get Started" button.

Micro & Macro Layouts

See the Pen Untitled by Mason Letourneau (@student147806) on CodePen.

Here we used a combination of grid and flex to create a job board page. There is a red border to show the layout of the page. We started by adding classes to all our HTML elements to organize ourselves. We then used CSS Grid to create the layout within the site. The header uses flex to space between, however. Finally, we created a media property to fix the layout if the screen was too small.
